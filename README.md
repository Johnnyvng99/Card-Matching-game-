A memory card matching game built in Python featuring adorable puppy photos. Flip cards to find matching pairs and test your memory!
How to Run
It is recommended to open the entire project folder in an IDE such as Visual Studio Code before running the game.

Open the project folder in VS Code (or your preferred IDE)
Run the main file:

"python project1.py"
Important: Photo Paths
The game loads puppy images from within the project folder. Make sure you open the entire project folder (not just project1.py by itself) so that the image file paths resolve correctly. 
If you run the script from a different directory, the photos may not load.

How to Play:
Enter your name when prompted
Choose a game mode:

8 cards — Easy
10 cards — Medium
12 cards — Hard


Cards are placed face-down on the board
Click a card to flip it and reveal a puppy photo
Find the matching pair for each card
Match all pairs to win!
